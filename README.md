=======


The software (version 1.0.0) associated with the manuscript "Rethinking Collinearity in Self-Organising Maps: Evidence from Geophysical Data Classification" for the calculation and storage of PSDs is licensed under MIT and published on GitHub.

==============


The repository includes:

1. Results of the simple synthetic model;
2. six complex synthetic models constructed using Noddy;
3. real world case study, where the data can be found in the repository below
   Xu, L., Green, E. C. R., McLean, M. A., \& Feltrin, L. (2025c). Applying Som Cluster Analysis with Iterative Refinement to Infer Lithology Units in Eastern Victoria, Australia - Code (1.0). In (Version 1.0.0) \[Computer Program]. Zenodo. https://doi.org/10.5281/zenodo.15502075 https://github.com/LXu1421/MultiStepLithologyCluster
4. a README.md file; and
5. a collection of Python scripts for sampling and unpacking the models in item 1,2, and 3.
   All codes and datasets are released under the MIT License.
